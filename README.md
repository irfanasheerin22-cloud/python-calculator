# 🧮 Python Calculator

A simple command-line calculator built using Python.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🚫 Prevents division by zero
- 🔄 Perform multiple calculations
- 🚪 Exit option

## 🛠️ Technologies Used

- Python 3

## 📂 Project Files

```
python-calculator/
├── calculator.py
└── README.md
```

## ▶️ How to Run

1. Open `calculator.py` in Pydroid 3 or any Python IDE.
2. Run the program.
3. Choose an operation.
4. Enter two numbers.
5. View the result.
